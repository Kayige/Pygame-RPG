# Pygame-RPG
My attempt at building a Final Fantasy Clone

## Basic Controls

**W** - Move Up<br/>
**S** - Move Down<br/>
**A** - Move Left<br/>
**D** - Move Right<br/>
<br/>
**M** - Toggles Menu<br/>
**SPACE** - Toggles Select<br/>
**ESC** - Quit Game<br/>
